Flampage - Unity3D

Top-Down shooter on PC


Nov 2016 – Dec 2016

Flampage is a 2 months long project produced in my first year at ENJMIN. Sole developper on the project, I worked on the following tasks

  • Programming on gameplay mecanics, including
    • Main Character (movements, dash, 360° shooting, damage handling, etc.)
    • Enemies IA which had 4 different behaviour (attacking the Monster, dash mecanism to avoid player bullets, following the player in case of damage and following exclusively the player)
    • Enemy waves spawing
  • File parser (.tsv) to be able to edit balancing values and enemy waves timing directly in a google sheet
  • Object pooling implementation for better perfomances
  • Assets integration
  • Creation of an algorithm to handle music depending on the game state (player's life, which stage the player is at)
