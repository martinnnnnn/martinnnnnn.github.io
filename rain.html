Rain Engine

3D game engine - C++/OpenGL

Rain Engine is a small project I am working on, and investing some time in whenever I get the chance. My goal is to create a 3D game with the engine when enough features are implemented and teach myself along the way. I am currently working on the 3D physics part of the engine. Things that are (mostly) working now are:

  • Window handling (Win32 API)
  • Game loop
  • Parsing configuration and scene files (Json)
  • Using a ECS library (entt) to handle world logic
  • 3D rendering with OpenGL (really basic for now)
  • 3D physics (applying forces, some collision detection and handling)
  • 3D Model loading (FBX SDK)
  • Shader loading
  • 2D text rendering

What I am currently working on:

  • Convex Mesh collision detection
  • Collision handling
