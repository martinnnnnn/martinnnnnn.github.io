Hindrance - Unity

March 2017 – June 2017

Hindrance is a group project made in 4 months at ENJMIN. It's a dance based game on tablet. Sole developper on the project, I worked on the following tasks:

  • Root Motion usage to improve the game feel for the dancer's movements.
  • Shader programming to render the character : glow shader for the light dots, modified post process motion blur applied only on the character to create light trails.
  • Shape recognition algorithme (vertical and horizontal lines, "bridge", loop going up or down, etc.) to allow control over the character by drawing on the screen.
  • UI flow integration
  • Sound assets integration using Wwise pluggin in Unity, to create interactive music during the dancing phases of gameplay.
  • Graphical assets integration.

Flampage - Unity

Nov 2016 – Dec 2016

Flampage is a 2 months long project produced in my first year at ENJMIN. Sole developper on the project, I worked on the following tasks:

  • Gameplay mechanics, including
    • Main Character (movements, dash, 360° shooting, damage handling, etc.)
    • Enemies IA which had 4 different behaviour (attacking the Monster, dash mecanism to avoid player bullets, following the player in case of damage and following exclusively the player)
    • Enemy waves spawing
  • File parser (.tsv) to be able to edit balancing values and enemy waves timing directly in a google sheet
  • Object pooling implementation for better perfomances
  • Assets integration
  • Creation of an algorithm to handle music depending on the game state (player's life, which stage the player is at)
